This page allows you to manage evaluations. You can download and print a blank evaluation form using the Download/Print Evaluation Form button. To add a new evaluation, click the Add Evaluation button.

An evaluator name and date are required for a new evaluation. When entering tryout numbers in the evaluation form, hit Enter after each number in in column, then hit Tab to move to the next column. The form will check to make sure that all tryout numbers on the form are valid before allowing you to save the evaluation.

Completed evaluations displayed on this page are collapsed by default and can be expanded to view the complete evaluation by clicking on the evaluator name and date. Once expanded, you can edit or delete the evaluation by clicking the appropriate button.
